\abstracteng{
    Turán-type problems are a central theme in extremal (hyper)graph theory,
    asking the maximum number of edges a $k$-graph $H$ can have, as a function of its number of vertices $n$,
    without containing a specific subgraph $G$.
    The maximum such number of edges is called the \emph{Turán number} and is denoted $\ex{n}{G}$.
    Any given $k$-graph $G$ has an associated constant $\pi(G)$ (which we call its \emph{Turán density})
    such that the Turán number $\ex{n}{G}$ is $(\pi(G) + o(1)) \binom{n}{k}$ as $n \to \infty$.
    This thesis deals with degenerate Turán problems (in which $\pi(G)=0$)
    by focusing on finding complete balanced $k$-partite $k$-subgraphs,
    denoted $\compoverset{k}{t}$, within a $k$-graph.

    Classical existence theorems by Kővari, Sós, Turán (for $k=2$) and Erdős (for $k \ge 2$)
    guarantee that $k$-graphs with constant edge density contain $\compoverset{k}{t}$ where $t$
    grows with the number of vertices $n$ (typically $t = \Omega\left((\log n)^{1/(k-1)}\right)$).
    These proofs are non-constructive,
    and locating such dynamically sized subgraphs efficiently is challenging as brute-force search becomes superpolynomial.

    This thesis presents a deterministic, polynomial-time algorithm that bridges this gap in the constant density case.
    Given a $k$-graph with $n$ vertices and $m$ edges, our algorithm finds a $\compoverset{k}{t}$ where $t$
    explicitly depends on $n$, $k$, and the density $d=m/n^k$, matching the $\Omega((\log n / \log(1/d))^{1/(k-1)})$
    scaling from existence proofs.
    Our method generalizes the bipartite case work by Mubayi and Turán, using a recursive strategy on link graphs.
    This provides an efficient, constructive proof for the existence of these growing
    $\compoverset{k}{t}$ structures, demonstrating they are algorithmically findable in polynomial time.
}
