\section*{Highlights}

\begin{itemize}
    \item We present a deterministic polynomial-time algorithm for dense $k$-uniform hypergraphs.
    \item The algorithm finds a complete balanced $k$-partite subgraph $K(t, \dots, t)$ efficiently.
    \item The part size $t$ matches the order from Erdős's non-constructive bound up to constants.
    \item Our method generalizes Mubayi and Turán’s approach from graphs to hypergraphs.
    \item The result provides a constructive counterpart to a classical extremal theorem.
\end{itemize}
